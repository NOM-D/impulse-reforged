impulse.Util

A generic module that holds anything that doesnt fit elsewhere

Functions

DrawBlur(panel, amount, passes, alpha)

 View source »

Blurs the content underneath the given panel. This will fall back to a simple darkened rectangle if the player has blurring disabled.

Parameters

  • panel panel

    Panel to draw the blur for

  • amount number default: 5

    Intensity of the blur. This should be kept between 0 and 10 for performance reasons

  • passes number default: 0.2

    Quality of the blur. This should be kept as default

  • alpha number default: 255

    Opacity of the blur

Example Usage

function PANEL:Paint(width, height)
    impulse:DrawBlur(self)
end

DrawBlurAt(x, y, width, height, amount, passes, alpha)

 View source »

Draws a blurred rectangle with the given position and bounds. This shouldn't be used for panels, see impulse:DrawBlur instead.

Parameters

  • x number

    X-position of the rectangle

  • y number

    Y-position of the rectangle

  • width number

    Width of the rectangle

  • height number

    Height of the rectangle

  • amount number default: 5

    Intensity of the blur. This should be kept between 0 and 10 for performance reasons

  • passes number default: 0.2

    Quality of the blur. This should be kept as default

  • alpha number default: 255

    Opacity of the blur

Example Usage

hook.Add("HUDPaint", "MyHUDPaint", function()
    impulse:DrawBlurAt(0, 0, ScrW(), ScrH())
end)

Client

Hooks that can be used in a Plugin or a Schema for the client

Functions

CLIENT:CheckMenuInput()

Called every tick, use this to check for key presses to open user interface elements (input.IsKeyDown)

CLIENT:CreateMenuMessages()

Called when the menu is active and MenuMessages are ready to be created

See Also

CLIENT:DefineSettings()

Called when you can define settings, all settings you want to define should be done inside this hook

See Also

CLIENT:DisplayMenuMessages()
Internal

This is an internal function! You are able to use it, but you risk unintended side effects if used incorrectly.

Called when the menu is active and MenuMessages are ready to be displayed

See Also

CLIENT:GetDeathSound()

Returns a custom death sound to override the default impulse one

Returns

  • string

    Sound path

CLIENT:PlayerGetJailData(endTime, jailData)

Called when the local player is sent to jail, provides jail sentence data

Parameters

  • endTime int

    When the jail sentence will end

  • jailData

    Data regarding the sentence including crimes commited

CLIENT:PostPopulateInventoryItemMenu(card, popup)

Called after the inventory item menu has been populated

Parameters

  • card

    The card panel that is used to display the item. You can use this to get the item data by calling card.Item

  • popup

    The dropdown menu that is used to show you actions you can do with the item, such as dropping it

Example Usage

hook.Add("PostPopulateInventoryItemMenu", "MyUniqueHookID", function(card, popup)
	    popup:AddOption("My Custom Action", function()
	        print("You clicked the custom action!")
	    end)
end)

CLIENT:PrePopulateInventoryItemMenu(card)

Called before the inventory item menu has been populated

Parameters

  • card

    The card panel that is used to display the item. You can use this to get the item data by calling card.Item

Returns

  • bool

    Should we allow the dropdown menu to be populated?

Example Usage

hook.Add("PrePopulateInventoryItemMenu", "MyUniqueHookID", function(card)
	    return LocalPlayer():IsAdmin() -- Only allow admins to interact with the item
end)

CLIENT:ShouldDrawHUDBox()

Called after a chat class message is sent
